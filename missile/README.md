# missile
基于[koa](http://git.caimi-inc.com/client/koa)的中后台项目模板
[wiki](http://git.caimi-inc.com/client/papaya/wikis/home)

## 环境依赖

- node 6+ LTS
- npm 3+

## 本地开发

```bash
npm i
npm start
```
浏览器访问 http://localhost:8080/missile
